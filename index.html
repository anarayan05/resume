TECHNOLOGY WORK AND PROJECTS

 

1.      Using Machine Learning to Recognize Names in Historic Newspaper Archives (Summer 2023 – till date): I created a program using Python and the Stanford Named Entity Recognition (NER) tool through Natural Language Toolkit (www.nltk.org) to download pages from Library of Congress (www.loc.gov) and identify person names from them so that they can be easily searched for individual names. In this project I worked with Prof. Haimonti Dutta (Haimonti Dutta - School of Management - University at Buffalo) who is an Associate Professor Management Science and Systems at University of Buffalo, School of Management.

 

2.      namanrush, a game (Fall 2022 – till date): I programmed a game using JavaScript, that is a single player casual game. It consists of a circle that a player can move using the arrow keys, and other circles that the player’s circles must avoid. If the player’s circle gets hit, it loses a life, the player’s circle has 3 lives. The player’s circle is also given 50 bullets they are released using the WASD keys for each direction. Along with avoiding the circles, the player’s circle must shoot all the circles in the field to advance to the next phase. This game is mostly done, but I am adding things here and there, and improving the game overall. The most recent major addition was making the other circles fight back against the user’s circle by shooting at it. This game is available to play here : anarayan05.github.io/namanrush/

 

3.      Math Flash Cards, a learning tool (2022): In the Summer of 2022, I programmed a learning tool, using JavaScript, that displays flash cards. When you click the card it exposes a simple addition, subtraction or multiplication problem. The audience is parents, who could use this for their young children learning mathematics fundamentals. Each problem dealt with whole numbers, and the solutions are whole numbers as well. After they are done each problem it will display, correct or incorrect, and then show a score out of 12.

 

4.      Joystick, a Robotic Toy Car (April 2020): In the 8th grade, I built a robotic toy car that uses ultrasound to detect objects in front of it and avoid them. I built this project using RaspberryPi, some stepper motors, an ultrasonic distance sensor and an H-Bridge. The programming was done using python inside Raspberry Pi. A short video explaining this project is available here: https://vimeo.com/520421124 . This project won Third Prize in the robotics category in the 7th Annual Student Showcase and Competition run by Buffalo State College.
